# 🎯 Welcome to My Face & Voice Recognition Attendance System Project

A smart attendance management system that uses Face Recognition and Voice Recognition to automatically identify students and mark attendance with high accuracy and security. The system uses dual biometric authentication to prevent proxy attendance and manual errors.

## 📌 Features
- Student registration with face image capture
- Face recognition using webcam
- Voice-based identity verification
- Automatic attendance marking
- Text-to-speech system notifications
- Training module for face data
- Attendance record storage
- User-friendly GUI

## 🧠 Working Principle
1. Student registers with personal details and face images.
2. System trains the face recognition model.
3. During attendance:
   - Camera detects and recognizes face
   - System asks student to speak their name/ID
   - Voice is converted into text
   - If face and voice both match, attendance is marked
4. Attendance is saved automatically.
